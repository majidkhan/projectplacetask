Shape Area Calculations

This is small app to calculate area of a selected shape. Local storage api has been used to develop this small app. No server side scripting is involved.

To calculate area of a shape select the shape from the menu items at left. Click the to perform calculation for each of the shape. The statistics block will display the statistics of the shape.
